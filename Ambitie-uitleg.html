Trots

Ik ben trots op mijn eigen ambitie project omdat ik met dit project de vrijheid had om te maken wat ik wilde.
Door het ambitie project ben ik er ook achter gekomen waar ik me meer op wilde gaan focussen,
het maken van games of websites.

Het maken van websites was voor mij het meest interesante en ben hier meer op gaan focussen,
Zo doende heb ik mijn shine project en vijfluik ook in de vorm van een website gemaakt.

Alsnog heb ik voor mijn ambitie project een 'Backpack' plugin gemaakt voor minecraft,
dit was een hele uitdaging omdat ik nog nooit met java had gewerkt, alles met code was redelijk nieuw voormij.
Ik ben trots hoever ik ben gekomen met mijn kennis die ik toen had, maar ik denk niet dat ik hier nog verder op in ga.
